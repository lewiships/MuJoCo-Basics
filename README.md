# MuJoCo Basics Workshop

## Pre-requisites

### 1) Install MuJoCo

Go to: https://mujoco.org/

Download the latest version of MuJoCo for your system:
- Windows (.zip)
- macOS (.dmg)

After downloading:

- Extract all
- Locate the folder called `bin`
- Inside this folder, find the application called `simulate`
- Run `simulate`

You should see a GUI window with:
> "Drag-and-drop model file here"

If you do not see this, your setup is not correct.

---

### 2) Download the Workshop Files

Download this repository:

- Option A: Clone using Git  
- Option B: Download as ZIP and extract

---

### 3) Test Your Setup (IMPORTANT)

- Open the `simulate` application
- Drag `task_1.xml` into the window

You should see:
- A box
- A sphere

If this works, you are ready for the session.

## MuJoCo Navigation Tips:
- Press **Ctrl + L** (Windows) or **Cmd + L** (Mac) to reload the model after editing
- Zoom with scroll wheel
- Rotate camera with left-click and drag
- Pan camera with right-click and drag
- Double left-click an object to make it the focus of other commands
- Double right-click an object to snap the camera to that object (very useful)
- Tab to hide/show the left control panel
- Shift + Tab to hide/show the right control panel
